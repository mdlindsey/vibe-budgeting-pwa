# Itemized Expense Tracker PWA

UI built with V0.dev

## Getting started

You will need a `.env` file with the following environment variables:

```
OPENAI_API_KEY="sk-proj-xxx"
GCP_SA_CREDENTIALS={...}
NEXT_PUBLIC_GCP_SA_EMAIL="your-sa-name@your-gcp-project.iam.gserviceaccount.com"
```
